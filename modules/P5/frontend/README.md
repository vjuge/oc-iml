## Stremalit 

`streamlit run app-streamlit.py`

`docker build -t p5front .`

### deploy on heroku

`$ heroku container:push web -a ociml-module5-frontend`

`$ heroku container:release web -a ociml-module5-frontend`

`heroku ps:scale web=1 -a ociml-module5`
